Downloading a file using powershell



First fix variables for easier access : $u , $p


$u="https://x-netrunner.github.io/storage/files/encrypter.py"


$p="C:/test.py"


Now you have 3 ways to download the file :



No. Method Url Destination Comment
1. Invoke-Webrequest -URI $u -Outfile $p $u $p mosty used
2. (New-Object System.Net.WebClient).DownloadFile ($u, $p) $u $p If your goal is to download a file, you will need to create a System.Net.WebClient object. When you create this object, you must call the DownloadFile method and provide values for the URL and for the output path.
3. tart-BitsTransfer -Source $u -Destination $p $u $p BITS refers to the Background Intelligent Transfer Service. This service is a part of the Windows operating system, and you must have BITS enabled for this technique to work.

Installing wsl on windows 11 or higher

This is actually very simple

steps:

No. steps comments
1. open powershell |wsl --install You will only need ubuntu for now if you want any other distro just download them from the store
2. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart enabling wsl on windows
3. wsl --set-default-version 2 there are 2 versions wsl1 and wsl2 -- we are setting wsl2 as default here
4. wsl --list --verbose | wsl --set-version [distribution name=""] [versionnumber][/versionnumber][/distribution] It should by default install ubuntu but if it doesn't use these commands or go to store and download ubuntu
